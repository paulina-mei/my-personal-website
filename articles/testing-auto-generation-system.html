Testing the Auto-Generation System

November 4, 2025

This is a test article to demonstrate our new auto-generation system. No more manual duplication!

How It Works

When you create an article, the system now automatically extracts all the data from the HTML file. This means you only maintain content in one place - the article HTML file itself.

The generate-search-data.js script scans all article HTML files, extracts the title, description, date, and content, then builds the search index automatically.

Key Benefits

  • No more content duplication - Edit once, propagates everywhere
  • Always in sync - Can't get out of sync anymore
  • Automatic date sorting - Newest articles appear first
  • Easy to update - Just edit HTML and regenerate
  • Single source of truth - Article HTML is authoritative

The Workflow

Creating or updating articles is now incredibly simple:

  1. Create or edit your article HTML file
  2. Run node scripts/generate-search-data.js
  3. Done! Your search index is updated

Or even better, use create-article.js which does everything automatically, including running the generator script for you.

The Result

A cleaner, more maintainable website that scales effortlessly. Article HTML files are now the single source of truth, and everything else is auto-generated from them.

This eliminates an entire class of maintenance issues and makes the website much easier to manage as it grows.
